Перетаскиваете ваш шрифт в Unity. 
Затем Window - Text Mesh Pro - Font Asset Creator. 
В верхнем поле выбираете шрифт и жмёте Generate Font Atlas и выбираете папку куда сохранить ассет. 
Если нужно с кириллицей, то в поле Character Set выбираем Unicode Range (Hex) 
и в поле ниже прописываем 0400-04ff, 
если нужно ещё с английскими буквами, символами и цифрами, то 0000-04ff
